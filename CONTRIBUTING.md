# Contribuindo para o Redesign do Ulife

Primeiramente, obrigado por se interessar em contribuir! Este é um projeto de estudo focado em melhorar a experiência do aluno (UX/UI).

## Como posso contribuir?

### Reportando Bugs
Se você encontrar um erro, abra uma **Issue** usando o template de "Bug Report" e descreva os passos para reproduzi-lo.

### Sugerindo Melhorias
Ideias para novos dashboards ou fluxos de UX são muito bem-vindas. Use o template de "Feature Request".

### Enviando Pull Requests
1. Faça um Fork do projeto.
2. Crie uma branch para sua modificação (`git checkout -b feature/MinhaMelhoria`).
3. Faça o Commit de suas alterações.
4. Envie para o GitHub (`git push origin feature/MinhaMelhoria`).
5. Abra um Pull Request descrevendo suas mudanças.

## Padrões de Código
* Utilize componentes funcionais e Hooks do React.
* Mantenha a tipagem correta no TypeScript.
* Siga o padrão de cores da paleta Zinc e Roxo estabelecida no Tailwind.
